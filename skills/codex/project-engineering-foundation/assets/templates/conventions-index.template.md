# Conventions 索引

> **目录定位**：项目特定约定的累积载体，在统一参考资产根 `ref/` 下，与 `ref/changelogs/` `ref/reviews/` `ref/plans/` 同级。
>
> **目的**：让项目 CLAUDE.md 保持**静态**（只放设计原则 + 流程性约定），动态累积的「反复出现过的设计决定」沉淀到本目录单独维护。
>
> **流程**：候选放 `tally.md`，count ≥ 3 走本仓库已配置的 review 流程后升级 → 新建 `<X>-<topic>.md`（X 递增整数）+ 本 INDEX 加行 + tally 删该候选条。
